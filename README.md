#ゲームについて
##ゲーム説明
>このゲームはフィールド内を自由に動き回ることの出来るゲームである．
<br><br>
##マニュアル
>ゲームを開始して，表1.1に示すキーボード入力を行う．

<br>
表1.1 キーボード入力による行動

| INPUT |   ACTION  |
|:---|:---|
|A|move left| 
|D|move right|
|SPACE|jamp|

##工夫した点
>マップの読み込みをJSONファイルから読み込むことで，ステージの変更を容易にした．<br>
>画像の分割読み込み．<br>
